# Chooz

A random selection web app featuring the Wheel of Names and more.

## Features

- Wheel of Names - Spin to randomly select a winner
- Coin Flip - Quick heads or tails decisions (coming soon)
- Dice Roll - Virtual dice for games (coming soon)

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React
- Vite
- Styled Components
- Framer Motion
